# 🤖 AI ToolKit

AI ToolKit is a web-based directory that helps users discover and explore the best AI tools across different categories. Instead of building AI models, the platform organizes popular AI tools based on their functionality, making it easy for users to find the right tool for their needs.

Whether you're looking for AI tools for writing, coding, video editing, audio processing, business, or productivity, AI ToolKit provides a clean and user-friendly interface to browse and access them.

---

## 🚀 Features

- 🔍 Discover hundreds of AI tools in one place
- 📂 Categorized tools for easy navigation
- 📝 Detailed description of each AI tool
- 🔗 Direct links to official websites
- 👤 User Registration and Login System
- 🎨 Responsive and modern UI
- ⚡ Fast navigation between categories
- 📱 Mobile-friendly design

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/SumitTodkar/AI-ToolKit.git
```

2. Move the project into the XAMPP `htdocs` folder.

3. Start **Apache** and **MySQL** using XAMPP.

4. Import the provided SQL database into phpMyAdmin.

5. Open your browser and visit:

```
http://localhost/AI-ToolKit/
```

---

## 🎯 How It Works

1. Users create an account or log in.
2. Browse AI tools by category.
3. View descriptions of each tool.
4. Click **Visit** to open the official website of the selected AI tool.
5. Explore different categories to discover tools for various tasks.
